# SwiftCart E-Commerce Project ЁЯЫТ

Welcome to **SwiftCart**, a modern and responsive e-commerce landing page built with Vanilla JavaScript, Tailwind CSS, and DaisyUI. This project fetches dynamic data from the Fake Store API to provide a seamless shopping experience.

## ЁЯФЧ Project Links
- **Live Demo:** [https://github.com/amimahedihasan/swift-cart-projects.git]
- **GitHub Repository:** [https://swif-cart-e-commerce.netlify.app/]
## ЁЯЪА Features
- **Dynamic Category Loading:** Categories are fetched directly from the API.
- **Category Filtering:** View products based on selected categories.
- **Product Details Modal:** Detailed view of each product including description and ratings.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Cart System:** Add/Remove items with a real-time count update (Challenge).

## ЁЯЫая╕П Technology Stack
- **HTML5**
- **Tailwind CSS** & **DaisyUI** (For Styling)
- **Vanilla JavaScript** (ES6+)
- **Fake Store API** (For Data)
